Командная строка Windows

Управление файлами и папками

  • COPY — Копирование файлов в другое место
  • DIR — Отображение файлов и папок в текущем каталоге
  • CD — Изменение каталога
  • MAKEDIR — Создание папки
  • MOVE — Перемещение файлов из одной папки в другую

Приложения и процессы

  • TASKLIST — Список выполняемых задач
  • TASKKILL — Остановка выполнения задачи (для остановки задачи используется PID, который можно узнать из TASKLIST)

Системная информация

  • DATE — Вывод или установка системной даты
  • TIME — Вывод или установка системного времени
  • HOSTNAME — Отображение имени компьютера
  • SYSTEMINFO — Отображение информации о конфигурации компьютера
  • VER — Просмотр версии Windows
